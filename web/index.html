ML CAPABILITIES

Each capability below is a trained model, served by the API, demoed here. Species identification is live today — this layout exists to grow.

Species Identification

Recognizes 15 species from the Swedish Leaf Dataset using a MobileNetV2-based classifier, with held-out test accuracy of 99.6%.

Instant Results

Upload a photo and get a prediction back in seconds, served by a lightweight FastAPI inference service.

Knows Its Limits

An out-of-distribution guard rejects photos that don't look like a clean leaf scan instead of guessing, so low-confidence results say so.

ABOUT THE PROJECT

An ML project first — this site is the demo layer.

Plantify's actual work is the training pipeline and the trained models in src/plantify/ — this website just shows what they can do. Each model is trained and evaluated with real held-out metrics (see the project's blog for how identification works), served through one API endpoint per capability, and demoed on one page per capability, free to try without an account.

Accurate

99.6% held-out test accuracy across all 15 known species, with a confidence floor to avoid false positives.

Honest

Flags uncertain or out-of-domain photos instead of forcing a guess onto one of its known species.

Open Source

Model, training pipeline, API, and this site are all in the public repository — nothing hidden behind a paywall.

No Sign-up Required

The identify feature works immediately — no account needed to try it.
