Instant Results

Knows Its Limits

About the project

An ML project first — this site is the demo layer.

Plantify's actual work is the training pipeline and the trained models in src/plantify/ — this website just shows what they can do. Each model is trained and evaluated with real held-out metrics (see the project's journal for how identification works), served through one API endpoint per capability, and demoed on one page per capability, free to try without an account.

Open Source

Model, training pipeline, API, and this site are all in the public repository — nothing hidden behind a paywall.

No Sign-up Required

The identify feature works immediately — no account needed to try it.

Self-Improving

A weekly job consults Pl@ntNet on photos the model doesn't recognize, stages good candidates, and retrains — see the journal.

Regression-Gated

Every automated retrain is accepted only if it doesn't regress overall or per-species accuracy — a worse model never ships silently.
